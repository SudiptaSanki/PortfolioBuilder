# Portfolio Builder — Open Source Template Gallery

A growing, open-source collection of **110 professionally designed portfolio templates** for every industry, profession, and tech stack. Our goal is to reach **1000+ templates** with the help of contributors from around the world.

---

## 🚀 Live Showcase

Browse all templates with live search, theme previews, and stack filtering at our showcase site (deployed on Vercel). You can filter by tech stack (**HTML/CSS/JS**, **React**, **Next.js**, **Vue**) or page structure (**Single-page** vs **Multi-page**).

👉 **[Launch Folio Hub Showcase Site](https://portfolio4builders.vercel.app/)**

---

## 📁 Template Directory Structure

Templates are organized with a deep, logical category structure:

```
templates/
  [category]/
    [subcategory]/
      [stack]/          ← html-css-js | react-vite | nextjs | vue-vite | etc.
        [theme-name]/   ← individual self-contained template folder
```

## 🛠️ How to Use a Template

1. **Clone or download** this repository:
   ```bash
   git clone https://github.com/SudiptaSanki/PortfolioBuilder.git
   ```
2. Navigate to your desired template directory (e.g. `templates/creative/photographer/html-css-js/minimal-monochrome/`).
3. Open `index.html` directly in your browser to inspect and preview locally.
4. Customize text, imagery, colors, and styling variables inside `style.css` and `script.js`.
5. Connect your contact form using free services like [EmailJS](https://emailjs.com/) or [Formspree](https://formspree.io/).
6. Deploy for free using [Vercel](https://vercel.com), [Netlify](https://netlify.com), or [GitHub Pages](https://pages.github.com/).

## 💻 Running Showcase Locally

The `showcase/` directory contains a Next.js web app that catalogs all templates with real-time search, interactive previews, and stack filters.

```bash
cd showcase
npm install
npm run dev
```

## 🤝 Contributing

We welcome contributions from developers, designers, and creators of all skill levels!

### How to Add a Template

1. **Fork** this repository.
2. Create your template folder under the appropriate directory structure:
   ```
   templates/[category]/[subcategory]/[stack]/[your-theme-name]/
   ```
3. Include your `index.html`, `style.css`, and `script.js` (or React/Vue/Next.js source files).
4. Register your metadata entry in `showcase/src/data/templates.json`.
5. Open a **Pull Request** detailing your template design and features!
