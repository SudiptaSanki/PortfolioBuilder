# UI/UX Designer Portfolio Template

A clean, minimalist, and soft portfolio template designed to showcase case studies and design processes.

## 🎨 Theme Details
- **Aesthetic:** Clean Minimal / Glassmorphism
- **Colors:** Soft Purples, Pinks, Teals, and White/Cream base
- **Fonts:** Outfit (Headings), DM Sans (Body)
- **Vibe:** Professional, approachable, design-centric

## ✨ Key Features
- **Glassmorphism:** Frosted glass effects on the navbar and UI elements.
- **Soft Gradients:** Beautiful, subtle gradient backgrounds and text fills.
- **Theme Toggle:** Built-in Light/Dark mode switcher (Dark mode utilizes deep purples).
- **Micro-interactions:** Smooth hover states on project cards and buttons.
- **EmailJS Ready:** Contact form pre-configured to send emails without a backend server.

## 📁 File Structure
- `index.html` - Home page featuring hero, services, and select case studies
- `about.html` - Biography, statistics, and a structured design process timeline
- `portfolio.html` - Grid of all design case studies and projects
- `contact.html` - Get in touch form and social links
- `css/style.css` - Core styling and theming
- `js/main.js` - Navigation logic, scroll effects, and form handling

## 🚀 Getting Started

1. **Edit Content:** Open the HTML files and replace the text and image placeholders.
2. **Change Colors:** Open `css/style.css` and adjust the variables in the `:root` section to match your personal brand colors.
3. **Configure Form:** 
   - Sign up at [EmailJS](https://www.emailjs.com/).
   - Add your `Public Key` to the `<head>` of `contact.html`.
   - Add your `Service ID` and `Template ID` in `js/main.js`.
4. **Deploy:** Upload the files to any static host like Netlify or GitHub Pages.
