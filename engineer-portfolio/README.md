# Software Engineer Portfolio Template

A sleek, tech-focused, and highly organized portfolio template inspired by modern developer tooling, terminals, and IDEs.

## 🎨 Theme Details
- **Aesthetic:** Modern Tech, Terminal Inspired
- **Colors:** Deep Navy (`#0a192f`), Slate (`#8892b0`), Cyan/Teal (`#64ffda`)
- **Fonts:** Inter (Body), Fira Code (Monospace Accents)
- **Vibe:** Highly technical, clean, structured, and developer-centric

## ✨ Key Features
- **Smart Navbar:** A navigation bar that hides when you scroll down to save screen space, and reappears as soon as you scroll up.
- **Tabular Archive:** The projects page features a clean table layout, perfect for listing numerous technical projects, hackathons, and open-source contributions.
- **Terminal Aesthetics:** Heavy use of monospace fonts for numbers, tags, and small text to give a terminal-like feel.
- **Hover Effects:** Subtle transform and shadow effects that make the UI feel responsive without being overwhelming.
- **EmailJS Ready:** Contact form pre-configured for serverless email functionality.

## 📁 File Structure
- `index.html` - Focused hero section and highlights of major projects
- `about.html` - Detailed biography and a dedicated monospace skills list
- `projects.html` - A comprehensive tabular archive of all technical work
- `contact.html` - Clean, terminal-styled contact form
- `css/style.css` - All styles, using CSS variables for easy theming
- `js/main.js` - Smart navbar logic, scroll reveals, and form handling

## 🚀 Getting Started

1. **Edit Content:** Open the HTML files and replace the dummy text with your actual experience. Be sure to link your GitHub and live URLs in the project cards.
2. **Personalize:** Change the theme colors in `css/style.css` if desired (the `:root` variables). A light theme is also included.
3. **Configure Form:** 
   - Sign up at [EmailJS](https://www.emailjs.com/).
   - Add your `Public Key` to `contact.html`.
   - Add your `Service ID` and `Template ID` in `js/main.js`.
4. **Deploy:** Drag and drop the folder into a static host like Netlify or Vercel.
