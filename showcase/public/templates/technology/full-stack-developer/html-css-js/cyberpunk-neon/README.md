# Developer Portfolio Template

A dark, cyberpunk-inspired portfolio template perfect for software developers, game devs, and tech enthusiasts.

## 🎨 Theme Details
- **Aesthetic:** Cyberpunk / Neon Dark
- **Colors:** Deep Dark Blue/Black, Neon Blue, Neon Pink
- **Fonts:** Space Grotesk (Headings), Inter (Body)
- **Vibe:** High-tech, energetic, modern

## ✨ Key Features
- **Typing Effect:** Dynamic typing animation in the hero section.
- **Theme Toggle:** Built-in Light/Dark mode switcher.
- **Scroll Animations:** Smooth reveal animations as you scroll down the page.
- **Responsive Navbar:** Desktop navigation that turns into a mobile hamburger menu.
- **EmailJS Ready:** Contact form pre-configured to send emails without a backend server.

## 📁 File Structure
- `index.html` - Main landing page with hero, skills, and featured projects
- `about.html` - Detailed background, experience timeline, and education
- `projects.html` - Full grid of project cards with filtering potential
- `contact.html` - Contact information and working email form
- `css/style.css` - All styling, variables, and responsive media queries
- `js/main.js` - Interactions, typing effect, and form logic

## 🚀 Getting Started

1. **Edit Content:** Open the HTML files and replace the dummy text, links, and image sources with your own data.
2. **Change Colors:** Open `css/style.css`. All colors are controlled by CSS variables at the top of the file in the `:root` pseudo-class.
3. **Configure Form:** 
   - Sign up at [EmailJS](https://www.emailjs.com/).
   - Add your `Public Key` in `contact.html` and `index.html`.
   - Add your `Service ID` and `Template ID` in `js/main.js`.
4. **Deploy:** Upload the contents of this folder to any static hosting provider (GitHub Pages, Netlify, Vercel).
