# Photographer Portfolio Template

A high-end, minimalist portfolio template designed for photographers and fine artists, featuring elegant typography and large image displays.

## 🎨 Theme Details
- **Aesthetic:** Minimalist, Editorial, Fine Art
- **Colors:** Monochrome (Black, White, Gray)
- **Fonts:** Playfair Display (Serif Headings), Lora (Serif Body)
- **Vibe:** Elegant, timeless, gallery-like

## ✨ Key Features
- **Automatic Hero Slider:** A fading image carousel in the hero section to cycle through your best shots.
- **Masonry Gallery:** A responsive, Pinterest-style masonry grid layout in the portfolio section.
- **Custom Lightbox:** Built-in vanilla JavaScript lightbox allowing users to click and expand images without leaving the page.
- **Theme Toggle:** Toggle between a pristine white gallery look and a sleek dark room look.
- **EmailJS Ready:** Contact form ready to receive client inquiries.

## 📁 File Structure
- `index.html` - Hero slider, selected editorial works preview
- `about.html` - Biography, artist statement, and exhibition history
- `portfolio.html` - Full masonry archive of photography work
- `contact.html` - Clean, minimal inquiry form and representation details
- `css/style.css` - All styling, including masonry CSS columns
- `js/main.js` - Slider logic, lightbox functionality, and form handling

## 🚀 Getting Started

1. **Upload Images:** Replace the Unsplash placeholder URLs with links to your own high-resolution photography.
2. **Masonry Configuration:** The masonry grid is built using pure CSS (`column-count`). You can adjust the number of columns across different screen sizes in `css/style.css` within the media queries.
3. **Configure Form:** 
   - Sign up at [EmailJS](https://www.emailjs.com/).
   - Add your keys to `contact.html` and `js/main.js`.
4. **Deploy:** Publish to any static hosting platform.
